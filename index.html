Expiry Label Printer

IoT web app running on a Raspberry Pi

Flask server running on a Raspberry Pi that acts as a WiFi access point. Mobile devices connect, open a browser, and print thermal expiry labels on a receipt printer via ESC/POS. Supports USB, network (TCP 9100), and RS-232 backends. Includes a web UI for managing product presets, configuring network interfaces, and switching printer connections — all without SSH.

  • ▸ RPi as WiFi AP — no router needed
  • ▸ ESC/POS thermal printing (USB, Net, RS-232)
  • ▸ Web-based product preset manager
  • ▸ Network interface config from the browser
  • ▸ systemd service — auto-starts on boot
  • ▸ Days/months expiry with calendar logic
Python Flask Raspberry Pi ESC/POS Linux / systemd NetworkManager
